About Me

In my position as an Installation Analysis Engineer at TechnipFMC I am currently specializing in lifting analysis. By analyzing 3D models of offshore structures, I distribute the physical properties and make a representative model in the software Orcaflex before performing a static and dynamic analysis. The goal of this analysis is to provide an Hs table which informs the vessel offshore of what conditions the offshore structure can be installed at.

In parallel, I work on developing automation scripts in Python to automate processes in the workflow.
